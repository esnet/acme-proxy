    Setup development environment#

    1. Install go >= 1.25
    2. Install pre-commit
    3. Clone the repo
    4. Run make dev

    Contribute#

    1. Fork the repo
    2. In your fork, create a new branch for your work
    3. Commit & push changes to your fork
    4. Submit a pull request
